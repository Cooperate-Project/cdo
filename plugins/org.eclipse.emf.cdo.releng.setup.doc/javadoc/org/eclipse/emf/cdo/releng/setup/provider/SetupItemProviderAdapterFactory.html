org.eclipse.emf.cdo.releng.setup.provider

Class SetupItemProviderAdapterFactory

  • java.lang.Object
    • org.eclipse.emf.common.notify.impl.AdapterFactoryImpl
      • org.eclipse.emf.cdo.releng.setup.util.SetupAdapterFactory
        • org.eclipse.emf.cdo.releng.setup.provider.SetupItemProviderAdapterFactory
  • All Implemented Interfaces:
    AdapterFactory, ComposeableAdapterFactory, IChangeNotifier, IChildCreationExtender, IDisposable

    public class SetupItemProviderAdapterFactory
extends SetupAdapterFactory
implements ComposeableAdapterFactory, IChangeNotifier, IDisposable, IChildCreationExtender
    This is the factory that is used to provide the interfaces needed to support Viewers. The adapters generated by this factory convert EMF adapter notifications into calls to fireNotifyChanged. The adapters also support Eclipse property sheets. Note that most of the adapters are shared among multiple instances.
