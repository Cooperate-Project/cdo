Interface TargletData

    public interface TargletData
extends EObject
    A representation of the model object 'Targlet Data'.

    The following features are supported:

    • Name
    • Roots
    • Source Locators
    • Repository Lists
    • Active Repository List
    • Active P2 Repositories
    • Include Sources
    • Include All Platforms
