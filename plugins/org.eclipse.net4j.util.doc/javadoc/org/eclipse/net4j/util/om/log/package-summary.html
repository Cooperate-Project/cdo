Package org.eclipse.net4j.util.om.log

Interfaces and classes for platform-independent logging.


Interface Summary
OMLogFilter Filters log events before they are being passed to the registered log handlers.
OMLogger The logger of a bundle.
OMLogHandler Handles filtered log events, for example appends them to a stream or the Eclipse error log.
 

Class Summary
AbstractLogHandler An abstract base implementation of a log handler that filters log events with a log level greater than a configurable maximum level.
EclipseLoggingBridge A log handler that appends log events to the Eclipse error log.
FileLogHandler A log handler that appends log events to a file.
Logger The default logger implementation.
OSGiLoggingBridge A log handler that appends log events to the OSGi log service.
PrintLogHandler A log handler that appends log events to error and output streams.
 

Enum Summary
OMLogger.Level Enumerates the possible severity levels of log events.
