Package org.eclipse.emf.cdo.compare

Client concepts for integrating EMF Compare with CDO.


Interface Summary
CloseableComparison A comparison that can be closed to dispose of used resources.
 

Class Summary
CDOCompareUtil Provides static factory methods that return CDO-scoped comparisons.
CDOCompareUtil.CDOComparison A comparison that can be closed to dispose of used resources.
CDOCompareUtil.CDOComparisonScope A CDO-specific base implementation of a comparison scope.
CDOCompareUtil.CDOComparisonScope.AllContents Takes an arbitrary object (including resource nodes) and returns matches for all elements of its content tree.
CDOCompareUtil.CDOComparisonScope.Minimal Takes a view/transaction and returns matches only for the changed elements of the entire content tree of its root resource.
CDOCompareUtil.CDOIDFunction An ID function that considers the CDOIDs of objects.
DelegatingComparison A comparison that delegates all methods to another comparison.
