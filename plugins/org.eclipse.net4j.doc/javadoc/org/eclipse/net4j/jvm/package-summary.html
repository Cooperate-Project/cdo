Package org.eclipse.net4j.jvm

Transport layer extension with support for JVM embedded connections.


Interface Summary
IJVMAcceptor An acceptor that implements JVM-embedded transport.
IJVMAcceptorManager A singleton that manages all JVM acceptors in the current JVM.
IJVMChannel A channel of a JVM connector.
IJVMConnector A connector that implements JVM-embedded transport.
 

Class Summary
